# Plain Language Reference Guide

Complete reference for clear writing principles, adapted from "A Plain English Handbook" (SEC, 1998) for universal application across languages.

> **Note on Language**: These principles originated from English writing guidelines but apply universally. Examples are provided in English and Chinese (中文) where patterns differ.

## Universal Principles by Language

### Active Voice

| Language | Passive Marker | Example (Before → After) |
|----------|---------------|-------------------------|
| English | "is/was/were" + past verb + "by" | "The report was written by her" → "She wrote the report" |
| Chinese | "被/由/为...所" | "报告被她写好了" → "她写好了报告" |

### Short Sentences

| Language | Target Length | Tips |
|----------|--------------|------|
| English | 15-20 words average | Avoid nested clauses; one idea per sentence |
| Chinese | 15-25 characters average | 避免长句套从句；避免过多的"之乎者也" |

### Personal Pronouns

| Language | First Person | Second Person | Note |
|----------|-------------|---------------|------|
| English | we/us/our | you/your | Creates direct engagement |
| Chinese | 我们/咱们的 | 你/您/您的 | "您" for formal contexts |

## The 13 Core Principles

### 1. Use Active Voice with Strong Verbs

- **Principle**: Readers understand active voice more quickly because it follows how we think.
- **Check for**:
  - English: Forms of "to be" + past tense verb + "by" phrases
  - Chinese: "被/由/为...所" + verb constructions
- **Fix**: Convert to active voice with specific, strong verbs.

**Examples:**

| Language | Before (Passive) | After (Active) |
|----------|-----------------|----------------|
| English | "The stock is bought by the investor." | "The investor buys the stock." |
| Chinese | "股票被投资者购买。" | "投资者购买股票。" |

### 2. Use Short Sentences

- **Principle**: Long, information-packed sentences are hard to understand.
- **Target**:
  - English: 15-20 words average; rarely exceed 30 words
  - Chinese: 15-25 characters average; avoid nesting multiple clauses
- **Fix**: Break long sentences into shorter ones; present one idea per sentence.

**Examples:**

| Language | Wordy | Concise |
|----------|-------|---------|
| English | "In order to facilitate the process of optimization, it is necessary that we first conduct a thorough analysis of the current system." | "To optimize, we first analyze the current system." |
| Chinese | "为了便于进行优化的过程，我们必须首先对当前系统进行一个彻底的分析。" (32字) | "要优化，我们先分析当前系统。" (11字) |

### 3. Use Personal Pronouns

- **Principle**: Pronouns clarify who the information applies to and create a direct tone.
- **Use**: "we/us/our" (first-person plural) and "you/your" (second-person singular).
- **Benefits**: Clarifies responsibilities, creates engagement, avoids abstractions, shortens sentences.

**Example:**

- Before: "This Summary does not purport to be complete and is qualified in its entirety..."
- After: "Because this is a summary, it does not contain all the information that may be important to you."

### 4. Bring Abstractions Down to Earth

- **Principle**: Concrete terms create mental images; abstractions don't.
- **Fix**: Use specific examples, especially with hypothetical investors or scenarios.
- **Hierarchy**: Asset → Investment → Security → Equity → Stock → Common stock → One share of IBM common stock

### 5. Omit Superfluous Words

- **Principle**: Remove words that don't add meaning.

**Common replacements:**

| English (Wordy → Simple) | Chinese (冗余 → 简洁) |
|-------------------------|---------------------|
| in order to → to | 为了……起见 → 为了 |
| in the event that → if | 在……的情况下 → 如果 |
| subsequent to → after | 在……之后的时间 → 之后 |
| prior to → before | 在……之前的时期 → 之前 |
| despite the fact that → although | 尽管……的事实 → 尽管 |
| because of the fact that → because | 由于……的原因 → 因为 |
| at this point in time → now | 在此时间点 → 现在 |
| for the purpose of → for | 为了……的目的 → 为了 |

### 6. Write in the Positive

- **Principle**: Positive sentences are shorter and easier to understand.
- **Fix**: Replace negative compounds with single words.

**Examples:**

| English | Chinese |
|---------|---------|
| "not able" → "unable" | "不是不能" → "可以" |
| "not accept" → "reject" | "不接受" → "拒绝" |
| "not many" → "few" | "不是很多" → "很少" |
| "not often" → "rarely" | "不常" → "很少" |
| "not...unless" → "only if" | "只有……才能" → "只要……就" |

**Chinese-specific:** Avoid double negatives like "不无道理" (not without reason) → "有道理" (reasonable).

### 7. Replace Jargon and Legalese

- **Principle**: Use short, common words instead of industry-specific terms.
- **Fix**: Use simpler synonyms; explain technical terms when first used.

**Examples:**

| English | Chinese |
|---------|---------|
| "terminate" → "end" | "终止" → "结束" |
| "elucidate" → "explain" | "阐明" → "说明" |
| "utilize" → "use" | "利用/使用" → "用" |
| "aforementioned" → (delete or use "this" or "that") | " aforementioned/前述的" → "这/那" |
| "leverage" → "use" | "leverage/撬动" → "利用" |
| "moving forward" → "from now on" | "moving forward/向前看" → "今后" |

### 8. Keep Subject, Verb, and Object Close Together

- **Principle**: Natural English word order is subject-verb-object.
- **Fix**: Move interrupting modifiers to separate sentences or sections.

### 9. Use "If-Then" Conditionals

- **Principle**: Conditional statements are clearer when following natural word order.
- **Rules**:
  - One if, one then: Start with if
  - One if, multiple thens: Start with if, tabulate the thens
  - Multiple ifs, one then: Start with then, tabulate the ifs
  - Multiple ifs and thens: Break into multiple sentences or use a table

### 10. Keep Sentence Structure Parallel

- **Principle**: Lists and series should use parallel parts of speech.
- **Fix**: Ensure all items in a list use the same grammatical form.

**Example:**

- Before: "to provide you with liquidity, protection of your investment, and high current income"
- After: "to provide you with liquidity, to protect your investment, and to generate high current income"

### 11. Avoid "Respectively"

- **Principle**: "Respectively" forces readers to reread and match items.
- **Fix**: State relationships explicitly.

**Example:**

- Before: "The Senior Notes and the guarantee...will constitute unsecured senior obligations of the Issuer and Island Holdings, respectively."
- After: "The senior notes are an unsecured senior obligation of the issuer, while the guarantee...is an unsecured senior obligation of Island Holdings."

## Additional Principles: Literal and Brief

Beyond the SEC's 11 principles, these two guiding concepts improve clarity:

### Spell It Out Literally

Don't use big vocabulary to hint at ideas. Just say exactly what you mean.

- **The problem**: Using complex words to imply meaning forces readers to guess your intent
- **The solution**: State your point directly, in the simplest words possible
- **The test**: If a reader had to read twice to understand, you hinted instead of told

**Examples:**

| Before | After |
|--------|-------|
| "The utilization of sophisticated methodologies may precipitate enhanced outcomes." | "Using better methods can improve results." |
| "We are not unmindful of the challenges inherent in this endeavor." | "We know this is difficult." |

### Brevity Is the Soul of Good Communication

Simple and clear beats fancy and smart-sounding. Every time.

- **The problem**: Fancy writing draws attention to the writer, not the message
- **The solution**: Write to inform, not to impress
- **The test**: Does this sentence sound like I'm trying to seem smart?

**Red flags for fancy writing:**
- Using rare words when common ones work
- Adding unnecessary qualifiers ("it is important to note that...")
- Complex sentence structures that delay the main point
- Academic or industry buzzwords that pad meaning

**Remember**: When writing articles to inform people, technical docs, or even just letters, any other way of writing feels like you are trying to be fancy just for the sake of seeming smart.

## Common Rationalizations (跨语言通用借口)

| English Excuse | Chinese Excuse | Reality |
|----------------|----------------|---------|
| "The original author is an expert" | "原作者是专家" | Expertise in content ≠ expertise in clear communication. |
| "Industry standards require this language" | "行业标准要求这样写" | Industry standards evolve. Plain language IS becoming standard. |
| "Short sentences sound choppy" | "短句显得断断续续" | Short sentences can flow well. Choppiness is a transition issue. |
| "Passive voice sounds more objective" | "被动语态更客观" | Passive voice often hides who did what—reducing accountability. |
| "We've always written it this way" | "我们一直都是这么写的" | "Always done this way" is not a defense of clarity. |
| "This sounds more professional" | "这样写更专业" | Professional ≠ complicated. Clear communication is professionalism. |
| "The audience is educated" | "读者都是有文化的" | Educated readers prefer clarity too. Complexity excludes. |

## Document Design Principles

### Typography

- Use serif fonts (e.g., Times, Garamond) for body text
- Use 10-12 point type size
- Avoid all caps (harder to read)
- Use bold and italics for emphasis, not underlining

### Layout

- Use left-justified, ragged-right text (not fully justified)
- Keep line length to 32-64 characters
- Use generous white space and line spacing
- Keep paragraphs short

### Organization

- Present the big picture before details
- Use descriptive headers and subheaders (avoid "General" or "Background")
- Group related information together
- Use bullet lists for complex material
- Use tables for data that readers need to compare
